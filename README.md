# EC2 Instances Setup

This repository contains information on the setup of development and production EC2 instances.

## Development Instance

### Instance Details
- AMI: Amazon Linux 2
- Instance Type: t2.micro
- Security Groups:SSH 
 


## Production Instance

### Instance Details
